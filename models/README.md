# Models Directory

This directory contains the trained weights and checkpoints for the Wound Assessment model.

## Final Selected Model
`SSL_IME_Round3_0.70_F1Moist.pth`

### Selection Reason
Round 3 with a `0.70` threshold was selected because it achieved the best **Moisture Macro F1** performance. While Round 4 produced more pseudo-labels, it did not improve moisture prediction performance.
